Localization with Particle Filter

Partner Project at Olin College of Engineering with K. Soltan

October 2018
Link to GitHub

K. Soltan and I created a particle filter to localize a Neato in a room, using ROS and python. You can see the documentation (including more lovely visualizations) in the README in our github repository, as well as all the code.
